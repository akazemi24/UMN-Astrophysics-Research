# UMN-Astrophysics-Research

**Project Summary:**  
Developed a Python-based alert system for real-time detection and notification of gravitational wave events in collaboration with the LIGO Scientific Collaboration (LSC). The system was designed to automate the identification of gravitational wave to allow rapid follow-up observations by astronomers.

**Repo:** [LIGO Alert System (Private, University of Minnesota Astrophysics Department)](https://github.com/mcoughlin/gwemlightcurves)
